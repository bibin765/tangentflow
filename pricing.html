Frequently asked questions

Is TangentFlow really free?

Yes. The core document builder is open source and runs entirely in your browser. There are no usage limits, no watermarks, and no sign-up required. The Pro and Enterprise tiers add cloud features for teams and businesses.

What is Pretext?

Pretext is an open-source JavaScript library created by Cheng Lou for precise text measurement and line-breaking. It measures every glyph mathematically using the browser's font engine, producing accurate text layout without DOM reflows. TangentFlow uses Pretext as its core text layout engine.

Do I need a server to generate PDFs?

No. Everything runs client-side in your browser using JavaScript. Pretext handles text measurement, and pdf-lib generates the PDF file — all without a server, headless browser, or any backend infrastructure.

Can I use this commercially?

Absolutely. The open-source version has no usage restrictions. Generate as many PDFs as you need for any purpose — personal, commercial, or enterprise.

How does this compare to Puppeteer?

Puppeteer renders HTML to PDF using a headless Chrome browser. It produces accurate results but requires a server, uses ~150MB of RAM, and takes 2-5 seconds per document. TangentFlow achieves comparable text wrapping accuracy in ~2ms, client-side, with no server dependency.

What about custom fonts?

The free version uses standard PDF fonts (Helvetica). Custom font support with font embedding is coming in the Pro tier. Pretext already supports measuring text in any font loaded in the browser.
